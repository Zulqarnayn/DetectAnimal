# DetectAnimal
This is an animal detection iOS application.

Recently I started learning Apple's ML and Vision Framework. Here I create an ML model using CreateML. Object Classes used here are

- **Butterflies**
- **Chickens**
- **Elephents**
- **Horses**
- **Spiders**
- **Squirells**
